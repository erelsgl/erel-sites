מלח1

קוד: מלח1 בתנ"ך

סוג: שזמש

מאת:

אל:

  • הגדרה_כללית: = שם "מלח", והנגזר ממנו, ועמו "בלויי מלחים" ו"מלוח". [שבי"ל]  
  • פועל: המלח (הפעל)  
  • עצם: מֶלַח (חומר)   
    = שם קיבוצי.  
  • תואר: מלֵחה (תואר)   
    = תואר לארץ [ש]לא תצמיח, וכמ"ש "ויזרעה מלח".  
  • פועל: מלַח (פעל)   
    = ענינו שימת מלח בדבר.  
  • חיה: מלוּח (עשב)  
  • כלי: מלחים (בגד)   
    = אולי הם בגדים הנוטים לריקבון, שנמלחים במלח לקיום.   [באה רק בספר ירמיהו]
  • פועל: נמלח (נפעל)   
    = ענינו שריפה וכליון, כמלח השורף, כמ"ש "גפרית ומלח שרפה כו'".  
  • צילום:   
  • מאמר: ים המלח / שדים / ערבה / קדמוני (הסבר לשמות) / אביתר כהן
  • מאמר: מאכלים מדבריים שנזכרו בתנ"ך: מן, שלו, עוגה, תמר, מלוח, מקורות מים / אבינועם דנין (קישור חיצוני)
