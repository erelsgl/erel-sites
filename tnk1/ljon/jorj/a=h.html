שורשים עם משמעות דומה, בחילוף בין אותיות א=ה

קוד: א=ה בתנ"ך

סוג: רבשורש

מאת:

אל:

  • עלשורש_אות: איכ=היכ 
    = שורשים עם משמעות דומה, בחילוף בין שתי אותיות קרובות.  
  • שורש1: ארמ=הרמ 
    = שם "ארמון", והוא בירה והיכל, ועמו "ההרמונה".   ארמון (מבנה)   הרמון (מבנה)  
  • שורש1: אתה=אתא 
    = ענינו ביאה.   אָתָא (פעל)   אָתָה (פעל)   ... אותיוֹת (שם מתואר)   איתון (מקום פרטי)   התה (הפעיל)  
  • שורש1: בטא=בטה 
    = ענינו דיבור קשה, לה"ר או שבועה.   בוטה (תואר)   בטָה (פעל)   ... ביטא (פיעל)   מבטא (קול)  
  • שורש1: דכא=דכה 
    = ענינו נמיכה ושפלות, ויש ["עד דכא" =] עד למות.   דַךְ (תואר)   דַכָּא (תואר)   ... דֹך (מקום)   דיכא (פיעל)   דכָה (פעל)   הִדַּכֵּא (התפעל)   נדכה (נפעל)  
  • שורש1: נכה=נכא 
    הֻכה (הפעל)   היכה (הפעיל)   מַכּוֹת (מקום)   ... מכָּה (מופשט)   נִכָּה (נפעל)   נֵכים (בעל תכונה)   ניכה (פיעל)   נכֵא (תואר)   נכֶה (בעל תכונה)  
  • שורש1: פלה=פלא 
    = ענינו הבדלה והפרשה, ועמו "הַפְלֵתִי".   הפלא (הפעיל)   הפלה (הפעיל)   ... נפלה (נפעל)   פֶלִאי (מלאך)   פילא (פיעל)   פלֵתי (בעל תפקיד)   פלוני (בעל תכונה)   פלמוני (בעל תכונה)  
  • שורש1: קרה3=קרא 
    = ענינו פגיעת דבר מבלי המחשבה עליו, ועמו "קרי".   הקרה (הפעיל)   לקראת- (מילת יחס)   ... מקרה (מופשט)   נקרה[פ] (נפעל)   קְרֵה (שם מתואר)   קֶרי (תואר פועל)   קרָה (פעל)  
  • מאמר: רפא = א. הבריא, ב. החליש את אחיזתו / אראל -> סגלות משלי
  • שורש1: שׂגא=שׂגה 
    = ענינו ריבוי וגודל.   השׂגה (הפעיל)   השׂגיא (הפעיל)   ... שׂגָה (פעל)   שׂגיא (תואר)   שׂגשׂג (פיעל)
