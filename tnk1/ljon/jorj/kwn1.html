כונ1

קוד: כונ1 בתנ"ך

סוג: שזמש

מאת:

אל:

  • הגדרה_כללית: = ענינו הכשר ותיקון, מוחשי או מופשט ["תכונה" בשורש תכנ]. [שבי"ל]  
  • פועל: הוכן (הפעל)  
  • פועל: הכין (הפעיל)  
  • פועל: התכונן (התפעל)  
  • תואר: כֵן[פ] (תואר פועל)   
    = נרדפת למילת "כה", גם לאמת דבר.  
  • תואר: כֵן[ת] (תואר)   
    = כמו נכון במוסרי [ישר וצדק], וחסר המתואר, או הוא תואר הפועַל.   [באה רק בספר בראשית]
  • פועל: כונֵן (פיעל)   
    = תיקון קיים ומתמיד.  
  • תואר: מוּכן (תואר)  
  • כלי: מכון (מבנה)  
  • כלי: מכונה (כלי)   
    = כלי למושב הכיור.  
  • פועל: נכון (נפעל)   
    = מוכשר ומתוקן.  
  • צילום:   
  • הבדל: כסא דוד / עמוס -> קובץ "הבדלה בין פסוקים דומים" ה'תשנ"ח-ט
  • מאמר: מה קרה בבית אבינדב? או: מה ההבדל בין נכון לכידון? / אבינדב אבוקרט -> כפית ה'תשס"ב ניסן
  • תוכן1: עד נכון היום / אראל -> סגלות משלי
