שורשים עם משמעות דומה, בחילוף בין אותיות ס=שׂ

קוד: ס=שׂ בתנ"ך

סוג: רבשורש

מאת:

אל:

  • מאמר: אפרים "ויאמרו לו אמר נא שבלת ויאמר סבלת" / רמי ניר
  • עלשורש_אות: גרס=גרשׂ 
    = ענינו כתיתה וטחינה.   גֶרֶשׂ (מופשט)   גרַס (פעל)   הגריס (הפעיל)  
  • עלשורש_אות: כסה=כשׂה 
    = שורשים עם אותה משמעות, בחילוף ס בשׂ.  
  • שורש1: מסר=משׂר 
    = ענינו נתינה, ועמו "המשׂור".   מַשּׂוֹר (כלי חיתוך)   מסַר (פעל)   נמסר (נפעל)  
  • עלשורש_אות: נסג=נשׂג 
    = שורשים עם אותה משמעות, בחילוף ס בשׂ.  
  • עלשורש_אות: סבכ=שׂבכ 
    = שורשים עם אותה משמעות, בחילוף ס בשׂ.  
  • עלשורש_אות: סהר=שׂהר 
    = שורשים עם אותה משמעות, בחילוף ס בשׂ.  
  • שורש1: סככ=שׂככ 
    הוסך (הפעל)   הסֵך (הפעיל)   מוּסך (הפעל)   ... מסָך (חומר בנין)   מסֻכָּה (שם מתואר)   משׂוּכָּה (שם מתואר)   סִכּוּת (כלי)   סָך (שם מתואר)   סֹך (מבנה)   סֻכָּה (מבנה)   סכַך (פעל)   שֹׂך (מבנה)   שֻׂכָּה (מבנה)   שׂוכך (פיעל)  
  • שורש1: סמר=שׂמר 
    = ענינו עמידת דבר מבלי נטות הנה והנה.   מסמֵר (כלי)   מסמַר (כלי)   ... משׂמֵר (כלי)   סימר (פיעל)   סמַר (פעל)  
  • עלשורש_אות: ספח=שׂפח 
    = ענינו חיבור ודיבוק, וע' "סיפח".   הסתפח (התפעל)   מספחת (שם מתואר)   ... משׂפח (שם מתואר)   נספח (נפעל)   סיפח (פיעל)   ספַח (פעל)   ספחת (מחלה)   שׂיפח (פיעל)  
  • שורש2: ספק=שׂפק 
    = שורש עם כמה משמעויות שונות.  
  • עלשורש_אות: סרפ=שׂרפ 
    = שורשים עם אותה משמעות, בחילוף ס בשׂ.  
  • עלשורש_אות: פשׂה=פסה 
    = ענינו התרחבות ותוספת, ועמו "פיסת בר".   פיסה (מופשט)   פשָׂה (פעל)  
  • עלשורש_אות: שׂטה=סטה 
    שטיה = סטיה (יציאה מהדרך) שהיא גם שטות (שיגעון) = ענינו נטיה והסרה.   סֵטים (שם מתואר)   ... שֵׂטים (שם מתואר)   שׂטָה (פעל)
