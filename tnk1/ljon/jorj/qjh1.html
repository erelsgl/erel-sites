קשה1

קוד: קשה1 בתנ"ך

סוג: שזמש

מאת:

אל:

  • הגדרה_כללית: = ענינו היפך רך, מוחשי או מופשט, ועמו "מקשה". [שבי"ל]  
  • פועל: הקשה (הפעיל)  
  • תואר: מקשֶה (תואר)   
    = תיקון השיער, שלא יהיו רכים ויתפזרו.  
  • תואר: מקשָה2 (שם מתואר)   
    = הרמב"ם ז"ל בפרק ג מהלכות בית-הבחירה, נראה שמפרש "מקשה" לומר שאינה חלולה, וע' שם בכסף משנה. שהוא קשה וחזק.  
  • תואר: נקשֶה (בעל תכונה)  
  • פועל: נקשה (נפעל)   
    = נלאה בצרות קשות.  
  • פועל: קישה (פיעל)   
    = ללידה.  
  • מופשט: קשִי (מופשט)  
  • תואר: קשֶה (תואר)  
  • פועל: קשָה (פעל)  
  • צילום:   
  • הגדרה: אלהים הוא אל-קושי, היה אומר נחום / רמי ניר
  • מאמר דמיוני: הקושי הוא עצום / רמי ניר
  • ביטוי הפעיל: הקשה לב   
    לב קשה, לב חזק, לב כבד, לב אמיץ, לב שמן
  • מאמר: פרשת בהעלותך - מקשה אחת ודקונסטרוקציה, חופש הביטוי ולשון הרע / חגי הופר
  • ביטוי בעל תכונה: קשה עורף   
    = שאינו יכול להסתובב לאחור, להביט במי שמוכיח אותו, ולשנות את דרכו הרעה
