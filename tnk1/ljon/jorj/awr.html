מאת:

אל:

  • הגדרה_כללית: = עיקר ענינו אור היפך חושך, והושאל לאור חכמה ושכל והזוהר, גם לאושר והצלחה. ועמו "אורות". [שבי"ל]  
  • מופשט: אוֹר[מ] (מופשט)   
    = כמו "חכמה", "שכל".  
  • מופשט: אוֹר[ע] (מראה)   
    = כל קרינה אלקטרומגנטית = נוגה, היפך חושך; ובלשון חז"ל גם לילה
  • פועל: אוֹר[פ] (פעל)   
    = ענינו הויית אור בנושא, ויהיה מתואר בו, כמו "חכמתי בחכמה", "גדלתי בגודל".  
  • מופשט: אוּר (מראה)   
    = אור הבא מן האש.  
  • כלי: אוּרים (כלי)   
    = ע' רש"י ורמב"ן.  
  • מופשט: אורה (מראה)   
    = "אור" בלשון נקבה.  
  • עצם: אורות (אוכל)   
    = מין ירק שמאיר את העיניים, יומא יח.  
  • פועל: האיר (הפעיל)   
    = ענינו על הרוב נותן אור, גם השמש והירח והנר נותני אור המה [והמקבל את האור בא בל' וב' ובמילות "אל" ו"על" ובלעדם], ויש אשר הפעל הוא הפעול, כמו "מחכים", "מגדיל".  
  • עצם: מאוּרה (גוף)   
    = העין, שהיא בעלת האור, ראב"ע ישעיהו יא8.  
  • מופשט: מאור[מ] (מופשט)   
    = מענין "האיר".  
  • עצם: מאור[ע] (עצם בשמים)  
  • פועל: נאור (נפעל)   
    = כמו "אור[פ]".
