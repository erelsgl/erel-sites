חיות כמשל

קוד: חיות כמשל בתנ"ך

סוג: משל

מאת:

אל:

  • ארי - נוהם, ודב - שוקק, מושל רשע על עם דל / אראל
  • דומה דודי לצבי או לעופר האיילים / אראל
  • והנה ארבע מרכבות יוצאות מבין שני ההרים / יהושע רוזנברג -> קול ישראל (קישור חיצוני)
  • חכמת הממשל במשל חכמה / יהודה אליצור -> עז לדוד - ספר בן גוריון (כתבי החברה לחקר המקרא בישראל ט"ו) (קישור חיצוני)
  • ככלב שב על קיאו - כסיל שונה באיוולתו / אראל -> סגלות משלי
  • לסוסתי ברכבי פרעה דימיתיך רעייתי / אראל
  • משלי עופות וחיות בר בספר הושע / אראל
  • סוסים ורוחות / אראל
  • פרה לימוד מדבר / אראל
  • פרשת ויחי - שליט הוא כמו אריה? / מוטי לקסמן
  • צאן יעקב כמשל לתלמידים ולעיקרי אמונה / שמוע הוד
  • קרבנות תמימים כמשל לאבות התמימים / חגי הופר
  • שארית יעקב בין הגוים / -> אתר אוניברסיטת בר אילן (קישור חיצוני)
  • שמעו את הדבר הזה פרות הבשן / יהושע רוזנברג -> קול ישראל (קישור חיצוני)
  • תנין - משל לכוחות הרוע, במיוחד מצרים / אראל
