אשור = רגל, צעד, דרך

קוד: אשור= בתנ"ך

סוג: הגדרה

מאת: אראל

אל:

אחת המשמעויות של המילה "אשור" בלשון המקרא היא - "רגל", ולכן המילה "אשורים" משמשת גם כמילה נרדפת ל"צעדים":

  • תהלים יז5: "תָּמֹךְ אשֻרי בְּמַעְגְּלוֹתֶיךָ, בַּל נָמוֹטּוּ פְעָמָי"
  • תהלים יז11: "אשֻרינו עַתָּה סבבוני[סְבָבוּנוּ], עֵינֵיהֶם יָשִׁיתוּ לִנְטוֹת בָּאָרֶץ"
  • תהלים מד19: "לֹא נָסוֹג אָחוֹר לִבֵּנוּ, וַתֵּט אשֻרינו מִנִּי אָרְחֶךָ"
  • תהלים לז31: "תּוֹרַת אֱלֹהָיו בְּלִבּוֹ, לֹא תִמְעַד אשֻריו"
  • תהלים מ3: "וַיַּעֲלֵנִי מִבּוֹר שָׁאוֹן מִטִּיט הַיָּוֵן, וַיָּקֶם עַל סֶלַע רַגְלַי כּוֹנֵן אשֻרי"
  • תהלים מד19: "לֹא נָסוֹג אָחוֹר לִבֵּנוּ, וַתֵּט אשֻרינו מִנִּי אָרְחֶךָ"
  • תהלים עג2: "וַאֲנִי כִּמְעַט נטוי[נָטָיוּ] רַגְלָי, כְּאַיִן שפכה[שֻׁפְּכוּ] אשֻרי"
  • משלי יד15: "פתי יאמין לכל דבר, וערום יבין לאשֻרו" = יבין את המשמעות של כל צעד שהוא עומד לעשות (פירוט).
  • איוב כג11: "באשֻרו אָחֲזָה רַגְלִי, דַּרְכּוֹ שָׁמַרְתִּי וְלֹא אָט"

בלשון התלמוד הבבלי יש מילה דומה - "אסתוירא" - שמשמעה "קרסול", החלק שבין השוק לבין כף-הרגל (ראו ערכין יט:, יבמות קג.).
מאותו שורש - א־ש־ר - בא גם הפועל "אִשֵּׁר", ויש לו כמה משמעויות אפשריות: הלך בדרך, הדריך אדם אחר בדרך, עשה אדם אחר מאושר, או אמר על אדם אחר שהוא מאושר. ישנם פסוקים שאפשר לפרש לפי יותר ממשמעות אחת:

  • משלי ט6: "עִזְבוּ פְתָאיִם וִחְיוּ, ואִשְרו בְּדֶרֶךְ בִּינָה" = 1. צעדו בדרך בינה, 2. היו מאושרים כאשר תלמדו בינה.
  • משלי ד14: "בְּאֹרַח רְשָׁעִים אַל תָּבֹא, וְאַל תְּאַשֵּׁר בְּדֶרֶךְ רָעִים" = 1. אל תלך בדרך רעים, 2. אל תחשוב שהרעים מאושרים.
  • ישעיהו א17: "לִמְדוּ הֵיטֵב, דִּרְשׁוּ מִשְׁפָּט, אַשרו חָמוֹץ, שִׁפְטוּ יָתוֹם, רִיבוּ אַלְמָנָה" = 1. הדריכו כל אדם שחמסו אותו, 2. עשו אותו מאושר.
  • ישעיהו ג12: "...עַמִּי מאַשריך מַתְעִים, וְדֶרֶךְ אֹרְחֹתֶיךָ בִּלֵּעוּ", ישעיהו ט15: "וַיִּהְיוּ מאַשרי הָעָם הַזֶּה מַתְעִים ומאֻשריו מְבֻלָּעִים" = 1. האנשים שאמורים להדריך את עם ישראל, מתעים אותו לדרך רעה, 2. נביאי השקר, שאומרים לעם שהוא מאושר והכל בסדר אצלו - מתעים אותו.
  • משלי כג19: "שְׁמַע אַתָּה בְנִי וַחֲכָם, וְאַשֵּׁר בַּדֶּרֶךְ לִבֶּךָ" = 1. הדרך את לבך בדרך טובה, 2. עשה את לבך מאושר בכך שתהיה חכם.
