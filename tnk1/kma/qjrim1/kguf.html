איברי גוף כמשל

קוד: גוף כמשל בתנ"ך

סוג: משל

מאת:

אל:

  • בטן = מקום ההולדה ומקום ההתחלה, מקום העיכול ומקום הרגשות המיידיים / אראל 
    כי נעים כי תשמרם בבטנך
  • גרון, גרגרת - משל לדיבור / אראל -> סגלות משלי
  • דם = משל לחיים של אדם שנרצח / אראל
  • העצמות היבשות - משל למדינת ישראל? / הלל אורי -> כפית ה'תשס"א תמוז
  • העצמות היבשות שקמו לחיים / יהודה איזנברג -> קול ישראל (קישור חיצוני)
  • חושים ומומים בספר ישעיהו / אראל -> לא גמור  
    השמן, הכבד, השע משל הספר החתום תור הזהב של העיוורים והחירשים
  • חיק = חיבוק, נשיאה, הסתרה / אראל
  • יד = בהשאלה: בליטה, צד, חלק, כוח / אראל
  • ידיו גלילי זהב - תפילין? / חגי הופר
  • יופיין של רגלי המבשר - משל ליופייה של בשורת הגאולה / יהודה איזנברג -> עיונים בספר נחום, אתר דעת (קישור חיצוני)
  • ימין ושמאל - משל לצדדים החזקים והחלשים באישיות / אראל
  • ימין, שמאל / אראל
  • כמים הפנים לפנים - כן לב האדם לאדם / אראל
  • לב חדש, רוח חדשה / אראל
  • לב מלא ברוח / אראל -> סגלות משלי 
    לב חדש, רוח חדשה גבה לב - גבה רוח רוח נכאה= תוכן ליבות - תוכן רוחות
  • מום - משל לפגם בהתנהגות / אראל 
    שיחת לו לא בניו מומם
  • משבר = א. גל שנשבר בים; ב. פתח הרחם שנשבר בזמן הלידה / אראל
  • משלים פיסחים - משלים של כסילים / אראל -> סגלות משלי
  • נופת תטופנה שפתי זרה, וחלק משמן חיכה / אראל
  • פרשת תרומה - משכן בגוף האדם / חגי הופר
  • שמאלו תחת ראשי וימינו תחבקני / אראל
  • שן רועה ורגל מועדת - מבטח בוגד ביום צרה / יעל, אראל
