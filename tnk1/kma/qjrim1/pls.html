פלס = מאזניים, משל לאיזון ולשקילת עדיפויות

קוד: פלס= בתנ"ך

סוג: הגדרה

מאת: אראל

אל: סגלות משלי

פֶלֶס הוא מכשיר לשקילה, דומה למאזניים, משלי טז11: "פֶּלֶס וּמֹאזְנֵי מִשְׁפָּט לה', מַעֲשֵׂהוּ כָּל אַבְנֵי כִיס" (פירוט).

מסתבר שהפלס גדול יותר מהמאזניים, ישעיהו מ12: "מִי מָדַד בְּשָׁעֳלוֹ מַיִם, וְשָׁמַיִם בַּזֶּרֶת תִּכֵּן, וְכָל בַּשָּׁלִשׁ עֲפַר הָאָרֶץ, וְשָׁקַל בַּפֶּלֶס הָרִים, וּגְבָעוֹת בְּמֹאזְנָיִם" - ה' שקל את ההרים בפלס הגדול, ואת הגבעות במאזניים הקטנים (מלבי"ם).

מהשם פֶלֶס בא הפועל פִּלֵּס (פילס), אך לגבי המשמעות שלו נחלקו המפרשים - יש אומרים שמשמעותו היא "שקל כמו במאזניים", ויש אומרים שמשמעותו היא "יישר כמו שמיישרים את לשון המאזניים":
  • ישעיהו כו7: "אֹרַח לַצַּדִּיק מֵישָׁרִים; יָשָׁר! מַעְגַּל צַדִּיק תְּפַלֵּס!" = הפסוק פונה לה' בתואר "ישר", ומבקש ממנו " שקול את המעגל = המנהג הצודק של הצדיק ותן לו את שכרו", או " יישר את המעגל = המסלול שהצדיק נוהג ללכת בו כך שיהיה חלק וללא מכשולים" (ראו מעגלים ישרים).
  • תהלים נח3: "אַף בְּלֵב עוֹלֹת תִּפְעָלוּן, בָּאָרֶץ חֲמַס יְדֵיכֶם תְּפַלֵּסוּן" = הפסוק מוכיח את הרשעים ואומר להם "כשאתם שוקלים במאזניים אתם עושים חמס (מאזניים מזוייפים)", או "אתם מיישרים את מעשי החמס שלכם ומציגים אותם כמעשי יושר וצדק".
  • תהלים עח50: "יְפַלֵּס נָתִיב לְאַפּוֹ, לֹא חָשַׂךְ מִמָּוֶת נַפְשָׁם, וְחַיָּתָם לַדֶּבֶר הִסְגִּיר" = כשה' היכה את המצרים, הוא שקל בצדק את עוצמת המכה והיקפה, היכה רק את בהמתם של המצרים ולא את הבהמות של ישראל; או - ה' סלל נתיב ישר עבור המכה, כך שתפגע רק במצרים ולא בישראל.
  • משלי ד26: "פַּלֵּס מַעְגַּל רַגְלֶךָ, וְכָל דְּרָכֶיךָ יִכֹּנוּ" = שקול ואזן היטב את מנהגיך, או יישר ונקה את דרכיך ממכשולים (פירוט).
  • משלי ה6: "אֹרַח חַיִּים פֶּן תְּפַלֵּס, נָעוּ מַעְגְּלֹתֶיהָ לֹא תֵדָע" = האישה הזרה מנסה לבלבל אותך כך שלא תוכל לשקול את דרכך בהגיון, ולא תוכל ליישר את דרכך ולחיות בצורה נכונה (פירוט).
  • משלי ה21: "כִּי נֹכַח עֵינֵי ה' דַּרְכֵי אִישׁ, וְכָל מַעְגְּלֹתָיו מְפַלֵּס" = ה' שוקל את כל מנהגיו של האדם, או - ה' מיישר את כל דרכיו של הצדיק ומנקה אותם ממכשולים (פירוט).
