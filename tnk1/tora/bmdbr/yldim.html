סיפורי ספר במדבר מותאמים לילדים

קוד: במדבר לילדים בתנ"ך

סוג: אוסף

מאת:

אל:

  • יא - בני ישראל מתלוננים במדבר / אראל
  • יג - משה שולח תיירים לארץ, והם מוציאים את דיבת הארץ רעה / אראל
  • יד - בני ישראל בוכים, וה' מעניש אותם שלא ייכנסו לארץ / אראל
  • טז - קורח ועדתו מורדים במשה, נשרפים ונבלעים באדמה / אראל
  • יז - בני ישראל מאשימים את משה שהרג את קורח ועדתו, ה' מביא עליהם מגיפה, ואהרן מציל אותם בקטורת / אראל
  • יח - מתנות הכהנים והלויים / אראל
  • כ - מרים מתה; משה ואהרן מכים את הסלע במקום לדבר אליו; מלך אדום לא נותן לבני ישראל לעבור; אהרן מת / אראל
  • כב - בלק מביא את בלעם ואתונו / אראל
  • כג - בלעם מנסה לקלל ולא מצליח / אראל
  • כד - בלעם פוקח את עיניו ומברך / אראל
  • כז - בנות צלפחד מבקשות נחלה בארץ ישראל / אראל
  • כז - צלפחד ובנותיו / אראל
  • לב - בני גד ובני ראובן מבקשים נחלה בעבר הירדן המזרחי / אראל
  • במדבר ט לילדים / אראל
  • במדבר כה לילדים / אראל
