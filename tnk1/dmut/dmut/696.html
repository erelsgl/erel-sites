נבל

מאת: רענן

קוד: נבל

 

  1. שמ"א כה 3: ושם האיש נבל ושם אשתו אבגיל והאשה טובת שכל ויפת תאר והאיש קשה ורע מעללים והוא כלבו (קרי: כלבי).

קבוצת-אב:   צאצאים אחרים של כלב  

פריטים אחרים בקבוצת האב:   יהדי  

אישה:   אביגיל  

  • מאמר: אין עושה טוב, אין גם אחד / יהושע רוזנברג -> קול ישראל
  • הבדלים: למה רצה דוד להרוג את נבל? / חגית מוריה -> שיעור ירושלים סב-סג
  • מאמר: נבל הכרמלי 1 / עדינה -> כפית כסלו ס"ב
  • הבדלים: נבל הכרמלי 2 / ספי פיקסלר -> כפית שבט ס"ב
  • מאמר: נבל הכרמלי 3 / יעקבי גולדשטיין -> כפית אביב ס"ג
  • כלל: קרבת המשפחה בין דוד לנבל / משה בלום -> שיעור ירושלים סב-סג
  • הגדרה: שכל טוב / אראל -> פו"ס משלי ב
  • הבדלים: שלוש גזיזות / אראל -> פו"ס 12
  • מאמר: ויהי כעשרת הימים ויגף ד' את נבל / יונתן קיוול, צמח יורה, חנן יריחובר, אראל סגל -> כפית סיון-אב נ"ח
