יונה

מאת: רענן

קוד: יונה נביא

 

  1. מל"ב יד 25: הוא (ירבעם בן יואש) השיב את גבול ישראל מלבוא חמת עד ים הערבה כדבר ה' א-להי ישראל אשר דבר ביד עבדו יונה בן אמתי הנביא אשר מגת החפר.
  2. יונה א 1: ויהי דבר ה' אל יונה בן אמתי לאמר.

אב:   אמתי  

  • כלל: ספר יונה - בין בריחה לבין ויכוח / הרב אלחנן סמט [סיכמה: פרידה נובוגרודסקי] -> יום עיון ישיבת הר עציון
  • מאמר: אדם בורח מייעודו / דוד נתיב -> חוג לומדי מקרא, אלון שבות
  • סיפור: היומן 1 / חגית מוריה -> כפית ס"א
  • מאמר: המבחן של יונה / חבי טרייסטר -> כפית כסלו ס"ב
  • כלל: המידות הדרושות כדי לשמוע את קול ה' / צחי -> דרישת השם באמצעות התורה שבכתב
  • מאמר: הקיקיון של יונה / יוסף צמודי -> בית מקרא צב, תשרי תשמ"ג
  • מאמר: ורחמיו על כל מעשיו (למשמעותו של ספר יונה) (1) / לאה פרנקל -> מעיינות
  • ספר: ספר יונה
  • כלל: יונה הנביא וחורבן נינוה / ר' קורמן -> פענוח פרשיות פרק יח
  • מאמר: יונה הנביא ומכניקה של נוזלים / אראל
  • מאמר: מדוע לא רצה יונה לנבא לאנשי נינוה? / מחברים שונים -> מקומות שונים
  • מאמר: נבואת יונה תשובת אנשי נינוה והוויכוח היהודי נוצרי / פרופ' אפרים א. אורבך -> מחניים ל"ג תשי"ח
  • מאמר: ספר יונה - האל והאדם, עבד מתפרץ, עבדים נאמנים, מעשה הקיקיון / שלמה דב גויטיין -> קטעים מתוך: אומנות הסיפור במקרא, "עיונים" למדריך ולמורה, הסוכנות היהודית, מח' לעליית ילדים ונוער, תשט"ז
  • כלל: ספר יונה - ספר הבריחה והשיבה / ישראל אלתר -> שיעור ירושלים סב-סג
  • מאמר: שיעור על יונה / חגית מוריה -> כפית תמוז ס"א
  • מאמר: שלוש בריחותיו של יונה / יהודה איזנברג -> אתר דעת
