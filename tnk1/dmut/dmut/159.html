לאה

מאת: רענן

קוד: לאה

 

  1. בראשית כט 16: וללבן שתי בנות שם הגדלה לאה ושם הקטנה רחל.
  2. בראשית כט 23: ויהי בערב ויקח את לאה בתו ויבא אתה אליו (יעקב) ויבא אליה.

אב:   לבן  

אחים מאב ואם:   רחל  

איש:   יעקב  

ילדים:   ראובן    שמעון    לוי    יהודה    יששכר    זבלון    דינה  

  • שיר: 'אוהב אותך לאה' - בביצוע צביקה פיק ומבצעים נוספים / צחור
  • מאמר לא גמור: הפעם אודה את ה' / פריס-סן-ז'רמן -> שאלה בלבד
  • מאמר דמיוני: השדה / עמוס -> הבדלה
  • הבדלים: לעלוקה שתי בנות / להב, אראל -> פו"ס 10
  • הגדרה: סגירת רחם / אראל
  • ציור: ציור בר כט 20 / סיגל (צילמה) -> תקרת קניון קסטרא
  • שיר: 'רחל' / אריאל ונגרובר
  • והנה בבוקר והנה את לאה / נתאלי שבח -> פייסבוק
