סגולות משלי פרק יט

קוד: סגלות משלי יט בתנ"ך

סוג: אוסף

מאת:

אל:

  • טעות בדרך נובעת משטחיות ולא מעונש / אראל -> סגלות משלי
  • הון יוסיף רעים רבים, ודל מרעהו ייפרד / אראל -> סגלות משלי
  • עד שקרים - יפיח כזבים / אראל -> סגלות משלי
  • שתי דרגות באיפוק / אראל -> סגלות משלי
  • שומר מצוה - שומר נפשו, בוזה דרכיו - ימות / אראל -> סגלות משלי
  • עצת ה' - בלב איש / אראל -> סגלות משלי
  • עדיף לא להבטיח, מאשר להבטיח ולא לקיים / אראל -> סגלות משלי
  • העצל מסתיר את כושר העבודה שלו / אראל -> סגלות משלי
  • עד בליעל יליץ משפט, ופי רשעים יבלע אוון / אראל -> סגלות משלי
