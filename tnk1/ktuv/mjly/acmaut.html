עצמאות כלכלית בספר משלי

קוד: עצמאות כלכלית בספר משלי בתנ"ך

סוג: אוסף

מאת:

אל: סגלות משלי

  • הלוואה = שיעבוד כלכלי / אראל -> סגלות משלי
  • לא לתת לאחרים לעבוד בשבילך / אראל -> סגלות משלי
  • ערבות = שיעבוד העתיד הכלכלי / אראל -> סגלות משלי
  • איך לא לחתום ערבות / אראל -> סגלות משלי
  • שימוש בכוח עבודה וחומרי גלם מקומיים / אראל -> סגלות משלי
  • עצמאות כלכלית חשובה יותר מכבוד / אראל -> סגלות משלי
  • גם כשעובדים אצל אחרים, יש לשמר את העסק העצמאי / אראל (הגהה: יעל) -> סגלות משלי
  • המזניח את העסק העצמאי שלו - עצל / אראל
  • נמלים - האם יש להן עצמאות כלכלית? / אראל -> סגלות משלי
  • ערבות - איך להינצל? / אראל -> סגלות משלי
  • עשיר ברשים ימשול, ועבד לווה לאיש מלוה / אראל
  • אל תהי בתוקעים כף, בעורבים משאות / אראל
