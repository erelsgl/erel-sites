סדר עדיפויות בספר משלי

קוד: סדר עדיפויות בספר משלי בתנ"ך

סוג: אוסף

מאת:

אל:

  • קביעת סדר עדיפויות - נושא מחקר עיקרי של שלמה / אראל
  • עדיף להשקיע באהבה בין הסועדים, מאשר באיכות הסעודה / אראל -> סגלות משלי
  • עדיף להשקיע ביחסי שלום ורעות בין המקריבים, מאשר בגודל הבית ובכמות הקרבנות / אראל -> סגלות משלי
  • עדיף להשיג מעט כסף ולעשות מעט צדקה, מאשר להשיג הרבה כסף לצורכי צדקה תוך פגיעה במשפט / אראל -> סגלות משלי
  • עדיף ללמד תורה לרבים מאשר לקיים מצוות פרטיות; עדיף לקיים מצוות מאשר ללמוד תורה לבד / אראל (שאלה: יעל) -> סגלות משלי
  • עדיף להיות עני עם ענוים מלהתעשר עם גאים / אראל -> סגלות משלי
  • לא לעשות חשבונות של סדרי עדיפויות במצב של פיתוי / אראל
  • מצוות בין אדם לחברו עדיפות על קרבנות / אראל -> סגלות משלי
  • עדיף לא להבטיח, מאשר להבטיח ולא לקיים / אראל -> סגלות משלי
  • עדיף להשקיע בחכמה ובינה מבכסף וזהב / אראל -> סגלות משלי
  • עדיף להשקיע בצבירת מוניטין מאשר בצבירת רכוש / אראל -> סגלות משלי
  • עדיף להשקיע בקשר עם ה' מבצבירת רכוש / אראל
  • עדיף ללכת לניחום אבלים מלחתונה / אראל
  • עדיף שרשעים יפסיקו להיות רשעים, מאשר שיביאו זבחים / אראל -> סגלות משלי
  • תורה לעומת תפילה / אראל -> סגלות משלי
  • טוב א מ-ב / אראל (הגהה: עופר לביא, יעל) -> סגלות משלי 
    הקשר בין אוכל לבין אהבה ושנאה הקשר בין קרבנות לבין שלוה ומרי... טוב מעט בצדקה, מרוב תבואות בלא... ... טוב שפל רוח את ענוים, מחלק שלל... עדיף לקנות חכמה ובינה מאשר כסף... עדיף להיות לבד מלהיות עם אשת מ... עדיף לקבל ביקורת מאשר לחיות בא... עדיף לא להבטיח, מאשר להבטיח ול... טוב רש הולך בתומו, מעיקש דרכיי... טוב רש הולך בתומו, מעיקש שפתיו... טוב מעט ביראת ה', מאוצר רב ומה... טוב שכן קרוב מאח רחוק
  • פלס = מאזניים, משל לאיזון ולשקילת עדיפויות / אראל -> סגלות משלי 
    אורח חיים פן תפלס כי נוכח עיני ה' דרכי איש, וכל ... פלס ומאזני משפט לה'
