סגולות משלי פרק יא

קוד: סגלות משלי יא בתנ"ך

סוג: אוסף

מאת:

אל:

  • בא זדון ויבוא קלון, ואת צנועים חכמה / אראל
  • התקוה האבודה / אראל -> סגלות משלי
  • צדיק מצרה נחלץ, ויבוא רשע תחתיו / אראל
  • שאול - איש תבונות / אראל -> סגלות משלי
  • הולך רכיל מגלה סוד, ונאמן רוח מכסה דבר / אראל -> סגלות משלי
  • חשבון נפש ברוב יועץ / אראל
  • גם מי שדואג לגופו הוא איש-חסד, וגם מי שפוגע בגופו הוא אכזר / אראל -> סגלות משלי
  • משרות מיותרות / אראל -> סגלות משלי
  • התמדה לעומת עקשנות, גמישות לעומת וותרנות / אראל
  • נזם זהב באף חזיר, אישה יפה וסרת טעם / אראל
  • צדיקים לא מקוים לרע, גם לא כדי שמתוך הרע ייצא טוב / אראל -> סגלות משלי
  • תפקידו העיקרי של השלטון - לדאוג שלציבור יהיה אוכל / אראל -> סגלות משלי
  • איך להתכונן לנפילה כלכלית / אראל -> סגלות משלי
  • להתעמק בניהול משק הבית / אראל (הגהה: יעל) -> סגלות משלי 
    חכמות נשים בנתה ביתה, ואיוולת ... עוכר ביתו ינחל רוח, ועבד אויל ...
