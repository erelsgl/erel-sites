סגולות משלי פרק לא

קוד: סגלות משלי לא בתנ"ך

סוג: אוסף

מאת:

אל:

  • אדם שעולה לשלטון צריך לרסן את תאוותיו / אראל
  • תנו שכר לאובד, ויין למרי נפש / אראל
  • לעזור לאנשים שמתקשים להתבטא במשפט / אראל -> סגלות משלי
  • אשת חיל יקרה מפנינים / אראל -> סגלות משלי
  • בטחון ואמון - תנאי יסודי לזוגיות טובה / מפרשים שונים
  • גמול על מעשים טובים ולא על מעשים רעים / אראל -> סגלות משלי
  • דורשי עבודה הם לא מובטלים / אראל (הגהה: יעל) -> סגלות משלי
  • ואת הסידורים האישיים יש לעשות לפני שעות העבודה / אראל -> סגלות משלי
  • זמן של עבודת כפיים הוא זמן טוב לתכנן איפה להשקיע את המשכורת / אראל
  • חשוב לשמור על כושר גופני - גם נשים! / אראל -> סגלות משלי
  • טוב לעבוד יותר קשה כשיש ביקוש לסחורה שלנו / אראל -> סגלות משלי
  • כל נזקק מקבל לפי צרכיו - כסף או חיבוק / אראל
  • להלביש את הבית בבידוד כפול / אראל
  • מעמדה הגבוה אינו מונע מאשת חיל לתפור את בגדיה בעצמה / אראל (הגהה: יעל)
  • נטלה על עצמה לפרנס את בעלה כי הוא עובד כשופט ואינו מקבל משכורת / אראל -> סגלות משלי
  • פתחה את פיה לדבר דברי חכמה וחסד, רק אחרי שהצליחה בחכמתה ועשתה חסד / אראל -> סגלות משלי
  • רק לשבח את האשה - לא לגנות נשים אחרות / אראל -> סגלות משלי
  • שטות היא לבחור בת-זוג על-פי נחמדות ומראה חיצוני בלבד; התכונה העיקרית שיש לבדוק היא יראת ה' / אראל -> סגלות משלי
  • תנו לאיש להגיש לאשתו מהמאכלים שבישלה / אראל
