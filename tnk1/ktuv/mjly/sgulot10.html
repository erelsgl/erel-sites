סגולות משלי פרק י

קוד: סגלות משלי י בתנ"ך

סוג: אוסף

מאת:

אל:

  • ללמד את הילדים איך להתעשר / אראל, יעל -> סגלות משלי
  • ילדים *חכמים* זה שמחה / אראל -> סגלות משלי 
    בן חכם ישמח אב, ובן כסיל תוגת ...
  • לא יועילו אוצרות רשע, וצדקה תציל ממוות / אראל
  • ביטוח מפני רעב לצדיקים / אראל
  • לא להרגיל את הידיים למרמה / אראל -> סגלות משלי
  • התנהגות משכילה בתקופות שפע / אראל -> סגלות משלי
  • זכר צדיק לברכה, ושם רשעים ירקב / אראל -> סגלות משלי
  • לקחת את המצוות ללב / אראל -> סגלות משלי
  • הולך בתום ילך בטח, ומעקש דרכיו יוודע / אראל -> סגלות משלי
  • קורץ עין ייתן עצבת, ואויל שפתיים יילבט / אראל -> סגלות משלי
  • מקור חיים פי צדיק, ופי רשעים יכסה חמס / אראל
  • שנאה - הגורם העיקרי לריב / אראל -> סגלות משלי
  • חכמים יצפנו דעת, ופי אויל מחתה קרובה / אראל -> סגלות משלי
  • פעולת צדיק לחיים - תבואת רשע לחטאת / אראל -> סגלות משלי
  • הסכנות שבדיבור / אראל -> סגלות משלי
  • להוכיח בקיצור / אראל -> סגלות משלי
  • עושר - ברכה או קללה / אראל -> סגלות משלי
  • ענישה מותאמת אישית / אראל -> סגלות משלי
  • יראת ה' תוסיף ימים / אראל
  • לשמוח בהמתנה / אראל
  • דרך ה' - להגן על התמימים ולשבור את עושי הרע / אראל -> סגלות משלי
