סגולות משלי פרק יד

קוד: סגלות משלי יד בתנ"ך

סוג: אוסף

מאת:

אל:

  • מבחן אישיות לעד / אראל -> סגלות משלי
  • ביקש לץ חכמה ואין, ודעת לנבון נקל / אראל (הגהה: עופר לביא) -> סגלות משלי
  • אל תחליטו בשביל אחרים מה טוב להם / אראל -> סגלות משלי
  • גם בשחוק יכאב לב, ואחריתה שמחה תוגה / אראל -> סגלות משלי
  • החכם נזהר שלא להיכנס לצרות / אראל -> סגלות משלי
  • להעביר את הדיונים בתביעות לעירו של הצדיק / אראל -> סגלות משלי
  • זהירות מיחס שלילי למכרים העניים / אראל -> סגלות משלי
  • הלוא יתעו חורשי רע, וחסד ואמת חורשי טוב / אראל -> סגלות משלי
  • עטרת חכמים עשרם. איוולת כסילים איוולת / אראל -> סגלות משלי
  • ברב עם הדרת מלך, ובאפס לאום מחתת רזון / אראל -> סגלות משלי
  • להרפות מהקנאה / אראל -> סגלות משלי
  • עושק דל - חירף עושהו, ומכבדו - חונן אביון / אראל -> סגלות משלי
