סגולות משלי פרק טז

קוד: סגלות משלי טז בתנ"ך

סוג: אוסף

מאת:

אל:

  • מלוכה משולבת ברוח ה' / אראל -> סגלות משלי
  • עצות למי שרוצה להיכנס לפוליטיקה בלי להתקלקל / אראל (הגהה: יעל) -> סגלות משלי
  • מי נותן לאדם את המילים? / אראל (הגהה: עופר לביא) -> סגלות משלי
  • להיות מודע להשגחה התמידית של ה' / אראל
  • ספר הכל לה' / אראל -> סגלות משלי
  • הגאוה פסולה אפילו במחשבה / אראל
  • איך לכפר על גאוה / אראל
  • שלום עם ה' / אראל -> סגלות משלי
  • טוב מעט בצדקה, מרוב תבואות בלא משפט / אראל -> סגלות משלי
  • חלוקת תפקידים בין ה' לאדם - תכנון וביצוע / אראל -> סגלות משלי
  • קסם על שפתי מלך = משל לשימוש באמצעים מתוחכמים, שנראים כמעשה כשפים / אראל -> סגלות משלי
  • חכמה - אפיק ההשקעה המשתלם ביותר / -> סגלות משלי
  • לפני כל אסון יש גאוה / אראל -> סגלות משלי
  • טוב שפל רוח את ענוים, מחלק שלל את גאים / אראל, יעל -> סגלות משלי
  • ריפוי בדיבור / אראל -> סגלות משלי
  • נפש עמל עמלה לו, כי אכף עליו פיהו / אראל -> סגלות משלי
  • איש תהפוכות ישלח מדון, ונרגן מפריד אלוף / אראל -> סגלות משלי
  • איך לזכות גם להורים וגם לנכדים / אראל (הגהה: יעל) -> סגלות משלי
  • טוב ארך אפיים מגיבור, ומושל ברוחו מלוכד עיר / אראל -> סגלות משלי
  • בחיק יוטל את הגורל, ומה' כל משפטו / אראל -> סגלות משלי
