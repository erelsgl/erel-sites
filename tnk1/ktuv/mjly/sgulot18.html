סגולות משלי פרק יח

קוד: סגלות משלי יח בתנ"ך

סוג: אוסף

מאת:

אל:

  • החכם מדבר כך שכולם יוכלו להבין בלי להתעמק / אראל -> סגלות משלי
  • לא לסלוח לרשעים / אראל -> סגלות משלי
  • כאבי בטן / אראל -> סגלות משלי
  • גם מתרפה במלאכתו, אח הוא לבעל משחית / אראל -> סגלות משלי
  • לרוץ בשם ה' / אראל -> סגלות משלי
  • לפני שאדם נשבר, ה' מגביה את ליבו; ולפני שאדם זוכה לכבוד, ה' מעמיד אותו בניסיון של עינוי והשפלה / אראל
  • לחשוב לפני שמדברים - כדי לא להגיד שטויות / אראל (הגהה: יעל) -> סגלות משלי
  • לא זקוקים? העבירו לנזקקים / אראל, יעל -> סגלות משלי
  • לראשון יש יתרון / אראל -> סגלות משלי
  • מדיינים ישבית הגורל, ובין עצומים יפריד / אראל -> סגלות משלי
  • התועלת שבלימוד שפות / אראל -> סגלות משלי
