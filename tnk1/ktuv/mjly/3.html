היחס לזולת בספר משלי

קוד: היחס לזולת בספר משלי בתנ"ך

סוג: אוסף

מאת:

אל:

  • אוילים - יליץ אשם; ובין ישרים - רצון / אראל
  • אל תאמר לרעך 'לך ושוב ומחר אתן' ויש אתך / אראל
  • אל תזלזלו באף אחד / אראל -> סגלות משלי 
    בז לרעהו - חוטא בז לרעהו - חסר לב בבוא רשע - בא גם בוז
  • אל תחליטו בשביל אחרים מה טוב להם / אראל -> סגלות משלי
  • אל תמנע טוב מבעליו, בהיות לאל ידך לעשות / אראל
  • אסור להסתיר פשעים בין אדם לחבירו / אראל -> סגלות משלי
  • האופי והרגשות שלנו משתקפים בלב הזולת / אראל
  • החובה להזדהות עם הזולת / אראל -> סגלות משלי
  • הצלת חיים - גם כשיש ספק / אראל
  • יחס אישי לכל בהמה / אראל -> סגלות משלי
  • לא להגיד הכל בפרצוף / אראל -> סגלות משלי
  • לא לקבל תרומות מרשעים / אראל -> סגלות משלי
  • להיזהר לפני ששופטים אנשים על מוזרות / אראל
  • לשפוט בצדק - בחכמה ותבונה ולא ברגש / אראל -> סגלות משלי
  • הצל לקוחים למוות, ומטים להרג אם תחסוך / אראל
  • לעג וזלזול / אראל
