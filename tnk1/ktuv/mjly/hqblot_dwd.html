משלי י-כב כמשל לבית דוד

קוד: משלי י-כב כמשל לבית דוד בתנ"ך

סוג: אוסף

מאת: מטמוניות למשפחות סופרים

אל:

  • י - דוד נבחר, אויביו נדחו / מטמונית למשפחות סופרים
  • יא / מטמונית למשפחות סופרים
  • יב / מטמונית למשפחות סופרים
  • יג - דוד ושלמה נבחרו, אבשלום ואדוניה נדחו / מטמונית למשפחות סופרים
  • יד - אמנון נדחה מהמלוכה / מטמונית למשפחות סופרים
  • טו - כלאב בן אביגיל סר מהמלוכה / מטמונית למשפחות סופרים
  • טז - שלמה בן בת-שבע נבחר למלך / מטמונית למשפחות סופרים
  • כ - בית יהוא נדחה, יואש-יאשיהו נבחרו / מטמונית למשפחות סופרים
  • כא - המלך נדחה או נבחר לפי יחסו לנביא / מטמונית למשפחות סופרים
  • כג - דוד מצווה את בניו ילידי חברון (למעט אדוניה) ואת שלמה / מטמונית למשפחות סופרים
  • כד - סוף הספר השני על סוף מלכות בית דוד / מטמונית למשפחות סופרים
