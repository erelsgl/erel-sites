סגולות משלי פרק כב

קוד: סגלות משלי כב בתנ"ך

סוג: אוסף

מאת:

אל:

  • נבחר שם מעושר רב, מכסף ומזהב חן טוב / אראל -> סגלות משלי
  • פגישה בין מעמדות / אראל -> סגלות משלי
  • ללמוד מטעויות של אחרים / אראל -> סגלות משלי
  • ענוה מביאה ליראת ה' ולנבואה / אראל -> סגלות משלי
  • הדרך משובשת - הבחירה בידך / אראל -> סגלות משלי
  • הלוואה = שיעבוד כלכלי / אראל -> סגלות משלי
  • זורע עוולה יקצור אוון, ושבט עברתו יכלה / אראל -> סגלות משלי
  • טוב עין - הוא יבורך כי נתן מלחמו לדל / אראל -> סגלות משלי
  • גרש לץ - ויצא מדון / אראל -> סגלות משלי
  • אהבה טהורה היא אהבה שמעצימה / אראל -> סגלות משלי
  • ה' שומר על השופט מטעות / אראל -> סגלות משלי
  • למרות שיש פיתויים ברחוב, לא צריך להסתגר בבית / אראל -> סגלות משלי
  • הפיצוי הראוי לדלים שנעשקו / אראל -> סגלות משלי
  • לא לגזול מצוות / אראל (הגהה: יעל) -> סגלות משלי
  • עוני מעודד גזל / אראל -> סגלות משלי
  • ערבות לעומת הלוואה / אראל -> סגלות משלי
  • חשיבות הגבול הבינלאומי / אראל -> סגלות משלי
  • איך להתייחס לאיש מהיר במלאכתו / אראל -> סגלות משלי
