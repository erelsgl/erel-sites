    אב ואם אהבת חכמה = תשוקה ללמוד ומחויי... אויל = שטחי איש מדינים = תמיד מוצא סיבה לר... אץ = חסר סבלנות ארך אפיים = מחכה זמן ארוך עד ש... אשה זרה ונכריה - משל ונמשל בער = טיפש כבהמה דל = חלש בגופו או ברכושו דעת = הכרות קרובה, באמצעות החו... זד = רע ומתגאה חוטא = מפספס, עושה רע מתוך טעו... חכמה = כישרון ללמוד וללמד חכמה בעיניו = חכמה מנקודת-מבט ... חנף = מנסה למצוא חן על-ידי ייפ... חסר-לב = לא מסוגל לחשוב בהגיון חרוץ = חד וזריז בתנועותיו ובמע... ישר = רודף שלום ואינו מכשיל את... כסיל = שונא חכמה בינה ודעת לץ = לועג ומזלזל בזולת מביש = לא יוצלח מזימה = תוכנית שמטרתה לפגוע ול... נבל = גס וברברי, חסר דרך-ארץ נדיב = אדם מורם מעם, המרים תרו... נלוז = עקום ונסתר נרגן = מאשים את הזולת בשנאה וכ... עז פנים = גורם נזק ללא הבעת בו... עיקש = עקום וקשה, לא-ישר ותוקפ... עני = שפל רוח, במעמד חברתי נמו... עצל = חסר רצון לעבוד ולהשקיע ל... ערמה = כישרון להטעות ולהיזהר מ... פתי = משתכנע בקלות צדיק = נותן לכל אחד את המגיע ל... ריק = נער-רחוב, ללא משפחה ורכו... רע - רשע רשע = עושה רע בשפע שכל = כישרון לתכנן ולהצליח במע... תבונה = כישרון להסיק מסקנות תהפוכות = חוסר-יציבות אידיאולו... תושיה = תכנית שמטרתה לעזור ולה... תמים = שלם ללא פגם מוסרי במעשי...
