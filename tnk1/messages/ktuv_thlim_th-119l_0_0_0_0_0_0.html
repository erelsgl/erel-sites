יפה שיחתן של עבדי אבות

קוד: שיחתן של עבדי אבות בתנ"ך

סוג: פירוש

מאת: אראל

אל:

אכן חז"ל אמרו "יפה שיחתן של עבדי אבות מתורתן של בנים", כי משיחתן של עבדי אבות אפשר ללמוד עקרונות של מוסר ודרך-ארץ שהם נצחיים וחשובים יותר מפרטי ההלכות שנזכרו בתלמוד.

כמובן, כדי להבין עקרונות אלה יש ללמוד בעיון.
