לאותם בני לוי שבחרו להיות "מורי שקר"...
להם, הקדיש הנביא מלאכי "תוכחת אמת"...

נאום מבוא:
"וְאַתֶּם סַרְתֶּם מִן-הַדֶּרֶךְ"
"הִכְשַׁלְתֶּם רַבִּים בַּתּוֹרָה"
"שִׁחַתֶּם בְּרִית הַלֵּוִי"
"אָמַר יְהוָה צְבָאוֹת"...


נאום פתיח:
"וְאָרוּר נוֹכֵל"
"וְיֵשׁ בְּעֶדְרוֹ זָכָר"
"וְנֹדֵר וְזֹבֵחַ מָשְׁחָת לַאדֹנָי"
"כִּי מֶלֶךְ גָּדוֹל אָנִי"
"אָמַר יְהוָה צְבָאוֹת"
"וּשְׁמִי נוֹרָא בַגּוֹיִם"...
(מלאכי א' 14)


נאום תוכחה:
"וְעַתָּה אֲלֵיכֶם הַמִּצְוָה הַזֹּאת--הַכֹּהֲנִים"
"אִם-לֹא תִשְׁמְעוּ"
"וְאִם-לֹא תָשִׂימוּ עַל-לֵב"
"לָתֵת כָּבוֹד לִשְׁמִי"
"אָמַר יְהוָה צְבָאוֹת"

"וְשִׁלַּחְתִּי בָכֶם אֶת-הַמְּאֵרָה"
"וְאָרוֹתִי אֶת-בִּרְכוֹתֵיכֶם"
"וְגַם אָרוֹתִיהָ"
"כִּי אֵינְכֶם שָׂמִים עַל-לֵב"

"הִנְנִי גֹעֵר לָכֶם אֶת-הַזֶּרַע"
"וְזֵרִיתִי פֶרֶשׁ עַל-פְּנֵיכֶם"
"פֶּרֶשׁ חַגֵּיכֶם"
"וְנָשָׂא אֶתְכֶם אֵלָיו"

"וִידַעְתֶּם"
"כִּי שִׁלַּחְתִּי אֲלֵיכֶם אֵת הַמִּצְוָה הַזֹּאת"
"לִהְיוֹת בְּרִיתִי אֶת-לֵוִי"
"אָמַר יְהוָה צְבָאוֹת"
"בְּרִיתִי הָיְתָה אִתּוֹ"
"הַחַיִּים וְהַשָּׁלוֹם"
"וָאֶתְּנֵם-לוֹ מוֹרָא וַיִּירָאֵנִי"
"וּמִפְּנֵי שְׁמִי נִחַת הוּא"
"תּוֹרַת אֱמֶת הָיְתָה בְּפִיהוּ"
"וְעַוְלָה לֹא-נִמְצָא בִשְׂפָתָיו"
"בְּשָׁלוֹם וּבְמִישׁוֹר הָלַךְ אִתִּי"
"וְרַבִּים הֵשִׁיב מֵעָו‍ֹן"
"כִּי-שִׂפְתֵי כֹהֵן יִשְׁמְרוּ-דַעַת"
"וְתוֹרָה יְבַקְשׁוּ מִפִּיהוּ"
"כִּי מַלְאַךְ יְהוָה-צְבָאוֹת הוּא"...

"וְאַתֶּם סַרְתֶּם מִן-הַדֶּרֶךְ"
"הִכְשַׁלְתֶּם רַבִּים בַּתּוֹרָה"
"שִׁחַתֶּם בְּרִית הַלֵּוִי"
"אָמַר יְהוָה צְבָאוֹת"

"וְגַם-אֲנִי נָתַתִּי אֶתְכֶם"
"נִבְזִים וּשְׁפָלִים--לְכָל-הָעָם"
"כְּפִי אֲשֶׁר אֵינְכֶם שֹׁמְרִים אֶת-דְּרָכַי"
"וְנֹשְׂאִים פָּנִים"
"בַּתּוֹרָה"...
(מלאכי ב' 9-1)
