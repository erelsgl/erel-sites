איש בשת - איש בעל - מות בעל

קוד: איש בשת - איש בעל - מות בעל בתנ"ך

סוג: מאמר

מאת: אביתר כהן

אל:

בישראל משום מה מתעלמים מהרעיונות של דיויד רוהל
שבין השאר טוען: ש לבאיה = שאול המלך, ובנו מותבעל הוא אישבעל / איש בושת {אגרות אל עמארנה }
באכדית: מות = איש, וגם בעברית מקראית
מתושלח האריך חיים יותר מכולם
בראשית ה27: "ויהיו כל ימי מתושלח תשע וששים שנה ותשע מאות שנה וימת"
ומשמעות שמו לא קשורה למוות אלא איש-שלח
דברים כו5: "וענית ואמרת לפני ה' אלהיך ארמי אבד אבי וירד מצרימה
ויגר שם ב מתי {= אנשי } מעט ויהי שם לגוי גדול עצום ורב"
דברים כח62: "ונשארתם ב מתי { אנשי } מעט
תחת אשר הייתם ככוכבי השמים לרב כי לא שמעת בקול ה' אלהיך"
אגב, בושת זה לא שם גנאי לאליל בעל, אלא שם נרדף
ירמיהו יא13: "כי מספר עריך היו אלהיך יהודה ומספר חצות ירושלם
שמתם מזבחות ל בשת  מזבחות לקטר ל בעל "
הושע ט10: "כענבים במדבר מצאתי ישראל כבכורה בתאנה בראשיתה ראיתי אבותיכם
המה באו בעל  פעור וינזרו ל בשת  ויהיו שקוצים כאהבם"
בעל (זכרות) = בושת = ערווה = יער
לכן קרית בעל = קרית יערים
בהרבה שפות יער = בוש
