(ב) נְטַעְתָּם גַּם שֹׁרָשׁוּ יֵלְכוּ גַּם עָשׂוּ פֶרִי
קָרוֹב אַתָּה בְּפִיהֶם וְרָחוֹק מִכִּלְיוֹתֵיהֶם:
