3) "כסה" בלוח לבנה הוא "אמצע החודש"
4) "לאחר שמונה ימים ביום שבת בעשרים ושלש לירח ניסן"..."וַיֶּאְסֹר אֶת רִכְבּוֹ"
5) "וכעבור ששה ימים בעשרים ותשעה לירח ניסן ביום ששי לשבוע פרעה הקריב"
6) "וביום השבת בראש החודש השני לאמור באחד לירח זיו"..."וַיֵּט משֶׁה אֶת יָדוֹ עַל הַיָּם"
7) "מתן תורה" התקיים בחמישה-עשר לחודש "סיון"

יחס "הפכפך" זה של הלוי (מר משה דין) אל התוכן המתועד בתנ"ך...הופך את יציאת מצרים להיות פלסתר...
