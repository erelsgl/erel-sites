כך זה עובד שם - שם המחבר של המאמר המקורי נמצא בהסטוריה, ושמות המחברים של התגובות נמצאים ליד התגובות.

בנוסף לכך, במערכת ויקי-יהדות, כל אחד יכול לערוך את המאמרים ולשנות אותם - זה אחד מתנאי השימוש באתר (זה לא אתר שלי).

כעת עליך להחליט, האם אתה מוכן שמאמריך יתפרסמו שם, ולהסתכן בכך שאנשים אחרים יבואו בעתיד ויערכו אותם? או שאתה מעדיף שהם לא יתפרסמו שם כלל?
