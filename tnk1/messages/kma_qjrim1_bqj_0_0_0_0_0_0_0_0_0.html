לידיעת כל המתעניינים באותה "תעודה" שציוונו (ישעיה ח' 16) ליצור:

א) רצף "טווחי הזמן" נקבע באותה "תעודה" אינו מקרי...

ב) מוכח מתוכן "תעודה" זו כי רק "מוצא פי-יהוה" הכתיב מראש את "טווחי הזמן"...

ג) רצף הזמנים "אמת" הוא רק כאשר "כוכב יעקב" יוצר חפיפה בין "לוח לבנה" (הלוח העברי שבנה נח)...לבין "לוח חמה" (הלוח הכללי שבנה יוליוס קיסר)...

ד) שני ה"לוחות" מתאחדים באותו יום של "פורים" הוא יום הולדתו של יוסף בן-יעקב...י"ג באדר...איחוד זה חוזר על עצמו כל 19 שנה...ואיחוד מיוחד מתרחש כאשר מופיע "כוכב יעקב" באותו יום של "פורים"...כל 76 שנה...אז מוכח כי בלעם היה נביא אלהים כי תיעד "דרך כוכב מיעקב" (במדבר כד' 17)...
