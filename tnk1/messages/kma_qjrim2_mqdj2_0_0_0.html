ואלה המידות (שאישר כורש מלך פרס) למקדש

מאת: רמי ניר

אל:

נכתב ב: 06:17:30  09.12.2008, כתוספת/תגובה ל: חיפשת בגוגל תמונות ?

המקדש השני נבנה עפ"י המידות המתועדות בצוו של דרייוש (בעזרתו של "דניאל"?) אחרי שחיפש ומצא בגנזך בית המלוכה את הצהרת כורש:

1. בֵּאדַיִן דָּרְיָוֶשׁ מַלְכָּא, שָׂם טְעֵם; וּבַקַּרוּ בְּבֵית סִפְרַיָּא, דִּי גִנְזַיָּא מְהַחֲתִין תַּמָּה--בְּבָבֶל.
2. וְהִשְׁתְּכַח בְּאַחְמְתָא, בְּבִירְתָא דִּי בְּמָדַי מְדִינְתָּא--מְגִלָּה חֲדָה; וְכֵן-כְּתִיב בְּגַוַּהּ, דִּכְרוֹנָה.
3. בִּשְׁנַת חֲדָה לְכוֹרֶשׁ מַלְכָּא, כּוֹרֶשׁ מַלְכָּא שָׂם טְעֵם--בֵּית-אֱלָהָא בִירוּשְׁלֶם בַּיְתָא יִתְבְּנֵא, אֲתַר דִּי-דָבְחִין דִּבְחִין וְאֻשּׁוֹהִי מְסוֹבְלִין; רוּמֵהּ אַמִּין שִׁתִּין, פְּתָיֵהּ אַמִּין שִׁתִּין.
4. נִדְבָּכִין דִּי-אֶבֶן גְּלָל, תְּלָתָא, וְנִדְבָּךְ, דִּי-אָע חֲדַת; וְנִפְקְתָא--מִן-בֵּית מַלְכָּא, תִּתְיְהִב.
5. וְאַף מָאנֵי בֵית-אֱלָהָא, דִּי דַהֲבָה וְכַסְפָּא, דִּי נְבוּכַדְנֶצַּר הַנְפֵּק מִן-הֵיכְלָא דִי-בִירוּשְׁלֶם, וְהֵיבֵל לְבָבֶל--יַהֲתִיבוּן, וִיהָךְ לְהֵיכְלָא דִי-בִירוּשְׁלֶם לְאַתְרֵהּ, וְתַחֵת, בְּבֵית אֱלָהָא

(עזרא פרק ו')
