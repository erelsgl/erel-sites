קבל את דבריהם! הסבר קצרצר

מאת: דביר

אל:

נכתב ב: 09:08:45  26.04.2006, כתוספת/תגובה ל: תגובה ל: תגובה ל: תגובה ל: תגובה ל: קבל את דבריהם!

דבר ראשון, התכוונת כנראה לרמב"ם בהקדמה לפרק חלק, שכותב שמי שמאמין שמדרשי חז"ל מתפרשים כפשוטם הוא סכל (דרך אגב, מיד אח"כ כתוב שמי שבגלל מה שכתוב במדרשים חושב שהם דברי הבל הוא אף סכל יותר... ועוד דברים שלא אכתוב כאן).
ובקשר לדיון, הרמב"ם ממשיך וכותב (מומלץ מאוד לעיין שם) שאותם הדברים שכתבו חז"ל במדרשים שאינם קיימים במציאות, הם משלים וחידות, "להבין משל ומליצה, דברי חכמים וחדותם". ועל כן אין להתייחס אליהם בזלזול אלא יש להעמיק בהם ולהבין את עומק מה שחז"ל רצו להעביר לנו.
בייחס לדברים שכן יכולים תיאורתית להיות במציאות, לדעתי הם נחלקים לשניים:
1. משלים וחידות, בדיוק כמו שכתב הרמב"ם לגבי הדברים הלא הגיוניים.
2. תאורי מציאות אמיתיים, שאם נעיין בזהירות, נמצא שהם באו ע"מ להסביר קושי כלשהו בתנ"ך, והם כעין פירושים על התנ"ך. צריך לעיין היטב כדי לקבוע איזה מדרש מתאים לאיזה סוג, ויכול להיות כמובן ששניהם נכונים...
מקווה שפתרתי את הענין.
