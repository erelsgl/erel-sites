די! מספיק! כל היום אתם רבים כמו ילדים בגן! אם אתם רוצים, תתכתבו באי-מייל. לידעתכם, מבקרי אתר הניווט בתנ"ך לא מעוניינים לדעת מי מכם יותר רשע, ואיך זה מוכח מפסוקים בתנ"ך (למרות שאני חייב להודות שזה די מבדר).
רמי ניר: רוב התענות שאתה מביא באלפי המאמרים שלך ממש לא מבוססות, ואתה כלכך אובססיבי עם הרעיון הזה שלך של "התעודה האלוהית", "אבן השתיה" ו"כוכב יעקב", שעד עכשיו לא ראיתי שום ביסוס משכנע לאף אחד מהרעיונות האלה (וקראתי הרבה מאמרים שלך). אז, או שתרד מהרעיונות האלה ותתחיל לכתוב מאמרים לעניין, או שחד וחלק תוכיח את התענות שלך בלי כל מני עובדות שאף אחד לא שמע אליהם, ובלי כל מני סימנים מעצבנים כמו " = ... ועוד. זה ממש לא תורם וסתם מבלבל!!!

מורה לתורה משה דין עבד יהוה מזרע אהרן הכהן הלוי: אותך אני פחות מכיר, כי אתה יותר חדש מרמי, וגם כי כמעט והפסקתי לבקר באתר מאז שרמי התחיל לפרסם דברים... לדעתי אתה קצת תופס מעצמך, עם כל התארים האלה שלך. אין כל כך מה לאמר לך, חוץ משאתה צריך להרגע קצת. תצא קצת מהבית, תלך לבלות. אתה נשמע כמו בנאדם מאוד לחוץ, כדי לך להשתחרר.
לסיכום אני מבקש מכם: בבקשה תפסיקו למלות את האתר הנפלא הזה בשטויות.
