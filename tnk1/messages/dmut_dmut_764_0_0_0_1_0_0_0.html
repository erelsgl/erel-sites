נאום מלאכי:
"כִּי מַלְאַךְ יְהֹוָה צְבָאוֹת הוּא"...
כלומר, "כהן" הוא שליח מצווה - "אדם" - "בשר" הוא...

לעומת זאת: כהן בוגד (גם באשתו) באלהים משקר וטוען כי בן אלהים הוא:
"וּמָה הָאֶחָד מְבַקֵּשׁ זֶרַע אֱלֹהִים"...
(מלאכי ב' 15)


לא רחוק הוא היום לדעת אמת - מי "מלאך" - ומי "בשר":
"הִנְנִי שֹׁלֵחַ מַלְאָכִי"
"וּפִנָּה-דֶרֶךְ לְפָנָי"...
(מלאכי ג' 1)
