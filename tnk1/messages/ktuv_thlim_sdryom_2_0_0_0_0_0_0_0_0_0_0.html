אני מקווה שעלי לא חל שום "אל טען כסיל",
כי אני לא חושב שאף אחד מכם הוא כסיל או אויל.
לשניכם יש בדרך כלל טיעונים טובים, מה שאני לא מבין למה אתם מתנצחים (בעצם בעיקר רועי מתנצח),
הדיון יכול להיות יותר פורה, אם ניקח אחד את השני ברצינות ונכבד את כל הדעות.
רועי, אני לא מבין למה ארנון נחשב בעיניך כאחד שאינו מאמין באמיתות התנ"ך. אני רואה את דעותיו דווקא כלגיטימיות.
למשל הרשב"א,מגדולי הפרשנים, מפרש על "ויהי ערב ויהי בקר" שהיום מתחיל בבוקר. וזה סותר את ההלכה היהודית.
אז נכון שראב"ע שרף את הפירוש (חפש בסידור "איש מצליח" או בגוגל "אגרת השבת"), אבל אפילו את ביקורת המקרא, הרב ברויאר למד, והפיק מזה תיאוריה מדהימה. (שיטת הבחינות). אז למה לנסות ישר לשלול את הדעות החדשות, אם אפשר ללמוד אותם לעומק ואז להגיב? (זה גם ימנע טעויות לא רצויות).
בתקווה לשיפור אווירת הדיונים באתר...
