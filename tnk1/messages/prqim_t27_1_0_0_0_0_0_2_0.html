אולם...עדיין לא מוזכר שמו של יום "הַשַּׁבָּת"...כמתועד:

"וַיִּסְעוּ מֵאֵילִם"
"וַיָּבֹאוּ כָּל-עֲדַת בְּנֵי-יִשְׂרָאֵל אֶל-מִדְבַּר-סִין"
"אֲשֶׁר בֵּין-אֵילִם וּבֵין סִינָי"
"בַּחֲמִשָּׁה עָשָׂר יוֹם לַחֹדֶשׁ הַשֵּׁנִי"
"לְצֵאתָם מֵאֶרֶץ מִצְרָיִם"...
(שמות טז 1)

לפי-כך...המסע המשיך באותו יום "שַׁבָּת"...לכוון רפידים...והגיעו עד "דָּפְקָה"...

כלומר...ביום ראשון (ט"ז באייר) בבוקר אכלו...פעם ראשונה "לחם שמים"..."בְּדָפְקָה"...

***

"וַיִּסְעוּ מִדָּפְקָה"
"וַיַּחֲנוּ בְּאָלוּשׁ"

"וַיַּחֲנוּ בִּרְפִידִם"
