"בְּיָמָיו עָלָה פַרְעֹה נְכֹה מֶלֶךְ-מִצְרַיִם"

"וַיִּקַּח עַם-הָאָרֶץ אֶת-יְהוֹאָחָז בֶּן-יֹאשִׁיָּהוּ"
"וַיִּמְשְׁחוּ אֹתוֹ וַיַּמְלִיכוּ אֹתוֹ תַּחַת אָבִיו"...
