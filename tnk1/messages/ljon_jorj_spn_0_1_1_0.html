אם אתה שואל אותי..."יַחַד שִׁבְטֵי יִשְׂרָאֵל"...כולל את שמעון...לעתיד לבוא...

פעם ראשונה..."יְהוָה מִסִּינַי בָּא וְזָרַח מִשֵּׂעִיר לָמוֹ--הוֹפִיעַ מֵהַר פָּארָן"...(דברים לג' 2)...משה השאיר לעתיד את הברכה לשמעון...

פעם שנייה...שמעון יקבל את הברכה:
"הִנֵּה אֲדֹנָי יְהוִה בְּחָזָק יָבוֹא"
"וּזְרֹעוֹ מֹשְׁלָה לוֹ"
"הִנֵּה שְׂכָרוֹ אִתּוֹ וּפְעֻלָּתוֹ לְפָנָיו"
"כְּרֹעֶה עֶדְרוֹ יִרְעֶה"
"בִּזְרֹעוֹ יְקַבֵּץ טְלָאִים"
"וּבְחֵיקוֹ יִשָּׂא עָלוֹת יְנַהֵל"...
(ישעיה מ' 11-10)
