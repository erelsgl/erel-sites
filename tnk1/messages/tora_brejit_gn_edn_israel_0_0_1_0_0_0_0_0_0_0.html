כאשר תתגשם הנבואה "מורה לתורה עבד יהוה" ידע להתבטא בשפת "צדק"...

אולם הנבואה עדיין לא התגשמה...לכן אתה הלוי אולי זכר ואולי נקבה (המתקרא משה דין) מתבטא כמו כל שקרן:
(ציטוט) "אך אתה אראל סגל וכל הולך בדרך רבני חזל וְהַמַּטִּים עַקַלְקַלּוֹתָם יוֹלִיכֵם יְהֹוָה אֶת פֹּעֲלֵי הָאָוֶן לַעֲזָאזֵל הַמִּדְבָּרָה שָׁלוֹם עַל יִשְׂרָאֵל ורק אם תשמור אֶת מִצְוֹת יְהֹוָה אֱלֹהֶיךָ לָלֶכֶת בִּדְרָכָיו וּלְיִרְאָה אֹתוֹ: כאשר מורה לך המורה לתורה עבד יהוה מזרע אהרן הכהן הלוי אז יטב לך בארץ הקודש (סוף ציטוט)

מה שמוכיח כי אינך "מורה"...כי אינך מסוגל לנקוט בצעד "ונקוטתם" (יחזקאל כ')...כך גם תביעתך ל"תרומות ומעשר" בטרם התגשמה הנבואה רק מוכיחה כי אתה "שליח לדבר עברה"...

אולי אתה "עבד של זדון"...
