כאשר "וַיֹּצִיאוּ דִּבַּת הָאָרֶץ"..."אֶרֶץ אֹכֶלֶת יוֹשְׁבֶיהָ הִוא"...(במדבר יג' 32)...שיקרו 10 המרגלים ביודעין...

כלומר, הוצאת דיבה...בשקר יסודה...

כמו-כן, מוציא הדיבה (השקרן) מוסיף גם הבעות פנים ושפת גוף להגדיל את "אפקט הדיבה"...ואכן היה "אפקט"...


לכן..."וַתִּשָּׂא כָּל-הָעֵדָה וַיִּתְּנוּ אֶת-קוֹלָם וַיִּבְכּוּ הָעָם בַּלַּיְלָה הַהוּא"...(במדבר יד' 1)


אז נתקבלה ההחלטה..."וַיֹּאמְרוּ אִישׁ אֶל-אָחִיו נִתְּנָה רֹאשׁ וְנָשׁוּבָה מִצְרָיְמָה"...(במדבר יד' 4)

כלומר..."אפקט הדיבה" השפיע ועיוות את השכל הישר...של "כָּל-הָעֵדָה"...
