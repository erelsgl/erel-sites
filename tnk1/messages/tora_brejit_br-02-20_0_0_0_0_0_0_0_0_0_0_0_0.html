כעת היום בשנת 5766 לספירה לגרוש האדם מגן-עדן
היום החמישה-עשר לחודש שנים-עשר הוא חודש אדר
יום ששי בשבוע
כעת עַל כֵּן קָרְאוּ לַיָּמִים הָאֵלֶּה פוּרִים
כַּיָּמִים אֲשֶׁר נָחוּ בָהֶם הַיְּהוּדִים מֵאוֹיְבֵיהֶם וְהַחֹדֶשׁ אֲשֶׁר נֶהְפַּךְ לָהֶם מִיָּגוֹן לְשִׂמְחָה וּמֵאֵבֶל לְיוֹם טוֹב לַעֲשֹוֹת אוֹתָם יְמֵי מִשְׁתֶּה וְשִׂמְחָה וּמִשְׁלוֹחַ מָנוֹת אִישׁ לְרֵעֵהוּ וּמַתָּנוֹת לָאֶבְיוֹנִים

וכעת אשלח לכם לאביונים מתנות למען תדעו כי איש אהל מועד אני אשר אלהי מוריש ומעשיר כי:
החכמה היא כֶּסֶף בעני כי בצל החכמה בצל הכסף
הדעת היא הנְחשֶׁת בנכה-הרוח
והבינה היא זָּהָב החרד אל דברי אשר דעת קדושים בו

וְאֶת כָּל אֵלֶּה יָדִי עָשָׂתָה וַיִּהְיוּ כָל אֵלֶּה נְאֻם יְהֹוָה וְאֶל זֶה אַבִּיט אֶל עָנִי וּנְכֵה רוּחַ וְחָרֵד עַל דְּבָרִי
אֵלֶּה פְקוּדֵי הַמִּשְׁכָּן מִשְׁכַּן הָעֵדֻת אֲשֶׁר פֻּקַּד עַל פִּי משֶׁה עֲבֹדַת הַלְוִיִּם בְּיַד אִיתָמָר בֶּן אַהֲרֹן הַכֹּהֵן
וְאֵלֶּה אֲשֶׁר הֶעֱמִיד דָּוִיד עַל יְדֵי שִׁיר בֵּית יְהֹוָה מִמְּנוֹחַ הָאָרוֹן וַיִּהְיוּ מְשָׁרְתִים לִפְנֵי מִשְׁכַּן אֹהֶל מוֹעֵד בַּשִּׁיר עַד בְּנוֹת אֶת בֵּית יְהֹוָה בִּירוּשָׁלִָם וַיַּעַמְדוּ כְמִשְׁפָּטָם עַל עֲבוֹדָתָם וְשִׁיר תְּהִלָּה וְהֹדוֹת לֵאלֹהִים יוֹמָם אֲשֶׁר יְצַוֶּה יְהֹוָה חַסְדּוֹ וּבַלַּיְלָה שִׁירֹה עִמִּי תְּפִלָּה לְאֵל חַיָּי
