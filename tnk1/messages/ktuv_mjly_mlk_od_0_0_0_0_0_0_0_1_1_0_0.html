אני רוצה שיראו את דברי כפי שהם נכתבו במקור ולא יוסיפו או יגרעו מהם מילה בשמי הקדוש כי כל איש ידבר בשם אלהיו לבדו
ואם אתה אראל סגל לא עומד בדבריך אשר אמרת מפורשות כי "שמות הכותבים יהיו בהיסטוריה" וכעת אתה אומר אחרת בגלל שלדבריך כך בנוי האתר אז:
תקשר את "שם הכותב" כקישור לכל הודעות דברי כתביו אשר כתב וגם על הקישור שיראה "שם הכותב" להראות בצבע וזאת למען להפריד ולהבדיל בין כותב אחד לבין כותב אחר כי יש כותבי אמת ויש כותבי שקר
ואין את דברי רוח חכמה לערב עם דברי כסילות
ואין את דברי איש הדעת לערב עם דברי הנבערים
כי כָּל הֵרַע אוֹיֵב בַּקֹּדֶשׁ וכי הכהן מבדיל בין דברי הקודש לבין דברי החול בין הטמא לבין הטהור
כי ידעתי את אשר כתבי דברי טהורים אמת קודש אנכי כותב בשם יהוה אלהי ישראל
