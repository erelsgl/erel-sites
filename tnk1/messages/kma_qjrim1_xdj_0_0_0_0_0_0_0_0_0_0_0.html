מורה לתורה משה דין
אני רק רוצה להוכיח לכם כי השנים המעוברות שהגו חזל הם לא אמת וכי מחזור 7 שנים מעוברות בתוך 19 שנה הוא בעליל אינו נכון
וכי יש ללכת על פי התורה ככתוב :
וְהַפִּשְׁתָּה וְהַשְּׂעֹרָה נֻכָּתָה כִּי הַשְּׂעֹרָה אָבִיב וְהַפִּשְׁתָּה גִּבְעֹל:
וְהַחִטָּה וְהַכֻּסֶּמֶת לֹא נֻכּוּ כִּי אֲפִילֹת הֵנָּה:
לאמור כִּי הַשְּׂעֹרָה אָבִיב = הַשְּׂעֹרָה בחודש האביב

אֱמֶת מֵאֶרֶץ תִּצְמָח
וְצֶדֶק מִשָּׁמַיִם נִשְׁקָף:
גַּם יְהֹוָה יִתֵּן הַטּוֹב וְאַרְצֵנוּ תִּתֵּן יְבוּלָהּ

כי אמת מארץ תצמח וארצנו תתן יבולה
כִּי הַשְּׂעֹרָה צמח הארץ עדות אמת על השנה על הראשון לחודשי השנה
וכאשר האביב פורח בעיצומו אז חג חודש האביב לאמור חג המצות בכסה ליום חגנו בחמשה עשר לירח ניסן

שמור את חודש הַשְּׂעֹרָה כִּי הַשְּׂעֹרָה אָבִיב
