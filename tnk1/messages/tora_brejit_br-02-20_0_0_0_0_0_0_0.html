בספר דברים פרק לג כתוב בנבואת משה לברכת בני ישראל:
(כה) בַּרְזֶל וּנְחשֶׁת מִנְעָלֶךָ וּכְיָמֶיךָ דָּבְאֶךָ:
ובספר דברים פרק ח כתוב:(ט) אֶרֶץ אֲשֶׁר לֹא בְמִסְכֵּנֻת תֹּאכַל בָּהּ לֶחֶם לֹא תֶחְסַר כֹּל בָּהּ אֶרֶץ אֲשֶׁר אֲבָנֶיהָ בַרְזֶל וּמֵהֲרָרֶיהָ תַּחְצֹב נְחשֶׁת" ומדובר על ארץ הצבי ארץ הקודש אשר יש בה ברזל ונחושת שהם בעיניך ולדבריך "אחד יותר רע מהשני"
