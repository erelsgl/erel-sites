רצף "טווחי הזמן" המתועד בתנ"ך אינו ניתן לזיוף...אינו ניתן לסילוף...ואינו מוטל בספק...

ואמנם באותה "אמת מדה" אשר "וַיַּאֲכִלְךָ אֶת-הַמָּן אֲשֶׁר לֹא-יָדַעְתָּ וְלֹא יָדְעוּן אֲבֹתֶיךָ"...(דברים ח' 3)
כל "טווחי הזמן" המתועדים בתנ"ך נקבעו במכוון:
"לְמַעַן הוֹדִיעֲךָ"
"כִּי לֹא עַל-הַלֶּחֶם לְבַדּוֹ יִחְיֶה הָאָדָם"
"כִּי עַל-כָּל-מוֹצָא פִי-יְהוָה"
"יִחְיֶה הָאָדָם"...
(דברים ח' 3)

"לָדַעַת אֶת-אֲשֶׁר בִּלְבָבְךָ הֲתִשְׁמֹר מִצְו‍ֹתָו--אִם-לֹא"...(דברים ח' 2)

רצף זה של "טווחי זמן"...הוא הנקרא "תעודה"...ואין לשנות להחסיר או להוסיף על "תעודה" זו...גם לא רגע אחד קטן...

אדם אשר מסלף...מוסיף...ומזייף את רצף "טווחי הזמן" המתועדים בתנ"ך...אינו אלא חוטא לאלהים...וכמובן נמנע הוא מלתת כבוד לאלהים...

במובן מסויים יכול אדם כזה אפילו "חכם"...להיות "שֵׂה תּוֹעֶה"...

ובמובן מסויים יכול הוא להיות "כמוך" (המתקרא הלוי משה דין)...זה אשר מנסה לגנוב את כבודו של אלהים...באופן קביעתו ה/תשס"ז...5767 שנה לבריאת העולם...
