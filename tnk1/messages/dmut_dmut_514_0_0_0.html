בלעם תיעד את שושלת (הבאים) הנולדים לאדם I ש"לא נולד" אלא נברא בדמות (בצלם) אלהים...

"שֵׁת" נולד והוליד את שושלת האצילים מאריכי הימים "בְּנֵי-שֵׁת" שהיו מסוגלים להוליד מאות שנים...

לעומת שושלת "בְּנֵי-שֵׁת" האצילים מאריכי הימים - נבראו בפרה-היסטוריה "זכר ונקבה" יצורי אנוש (מאיכות שלא האריכה ימים) ואותם היפרו הבאים לשושלת "בְּנֵי-שֵׁת" ושיבחו את המין האנושי...

בלעם הוכיח בהגדרה זו ידע שרק אלהים יכול היה לגלות לנביאו. בלעם רצה שהתיעוד הזה יזכר לנצח...

גם קין (בראשית ד' 12) לקח אשה שלא הייתה אחותו משושלת אדם. קין ואשתו הולידו את בנם חנוך שבנה עיר היסטורית ראשונה וקרא לה "חנוך" ובה ישבו תושבי "ארץ נוד קדמת עדן"...
