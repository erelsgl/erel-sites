נדגלות = עין גנים?

קוד: נדגלות= בתנ"ך

סוג: פרטים1

מאת: מטמוניות למשפחות סופרים (schriber @ 012.net.il)

אל:

שלמה המלך אומר בשיר השירים פרק ו כך :

( ב ) דּוֹדִי יָרַד לְגַנּוֹ לַעֲרוּגוֹת הַבֹּשֶׂם לִרְעוֹת בַּגַּנִּים וְלִלְקֹט שׁוֹשַׁנִּים :

( ד ) יָפָה אַתְּ רַעְיָתִי כְּתִרְצָה נָאוָה כִּירוּשָׁלָם אֲיֻמָּה כַּנִּדְגָּלוֹת :

( ה ) הָסֵבִּי עֵינַיִךְ מִנֶּגְדִּי שֶׁהֵם הִרְהִיבֻנִי שַׂעְרֵךְ כְּעֵדֶר הָעִזִּים שֶׁגָּלְשׁוּ מִן הַגִּלְעָד :

( י ) מִי זֹאת הַנִּשְׁקָפָה כְּמוֹ שָׁחַר יָפָה כַלְּבָנָה בָּרָה כַּחַמָּה אֲיֻמָּה כַּנִּדְגָּלוֹת :

מסקנות :

1) תרצה וירושלים הם ערים , מכאן שנגדלות ג " כ מכוונת לעיר מסויימת

2) התואר " יפה " מיוחס לתרצה וללבנה , מכאן שירושלים " כחמה ", ונדגלות כצבאות השמים , כוכבים מזלות וגלגלי הרקיע

3) מסעיף 2 נסיק שתרצה ונדגלות ( לבנה וכוכבים ) היו בקרבת מקום .

4) נגדלות הטילה אימה מרהיבה על רואיה – במיוחד בעיניה ( פסוקים ד ה )

5) מסעיף 4 נסיק שב " נדגלות " מסתתר " גן " " גנות ", ויחד עם " עין ", מתקבל " עין גנים " שבנחלת יששכר , ע " ש מעיינות גנים שהיו בה ( יש משערים את מקומה הגיאוגרפי בסביבת ג ' נין בין תרצה לשומרון , משוכה צפונה להם , במורדות הרי שומרון בואכה עמק יזרעאל ).

( יהו ' יט יז ) לְיִשָּׂשכָר יָצָא הַגּוֹרָל הָרְבִיעִי לִבְנֵי יִשָּׂשכָר לְמִשְׁפְּחוֹתָם :... ( כא ) וְרֶמֶת וְעֵין גַּנִּים וְעֵין חַדָּה וּבֵית פַּצֵּץ :


6) נראה לומר שמקומה של הרעיה היה בעיר " עין גנים " ( פסוק ב ), וכמו שכתוב בפרק ד :

( יב ) גַּן נָעוּל אֲחֹתִי כַלָּה גַּל נָעוּל מַעְיָן חָתוּם :

( טו ) מַעְיַן גַּנִּים בְּאֵר מַיִם חַיִּים וְנֹזְלִים מִן לְבָנוֹן :

( טז ) עוּרִי צָפוֹן וּבוֹאִי תֵימָן הָפִיחִי גַנִּי יִזְּלוּ בְשָׂמָיו יָבֹא דוֹדִי לְגַנּוֹ וְיֹאכַל פְּרִי מְגָדָיו :


7) יתכן שעין גנים נקראה גם " בית הגן " ע " ש ביתה של הרעיה , כמו שמרומז בפרק ח :

( ב ) אֶנְהָגֲךָ אֲבִיאֲךָ אֶל בֵּית אִמִּי תְּלַמְּדֵנִי אַשְׁקְךָ מִיַּיִן הָרֶקַח מֵעֲסִיס רִמֹּנִי :

( יג ) הַיּוֹשֶׁבֶת בַּגַּנִּים חֲבֵרִים מַקְשִׁיבִים לְקוֹלֵךְ הַשְׁמִיעִינִי :

( מ " ב ט כז ) וַאֲחַזְיָה מֶלֶךְ יְהוּדָה רָאָה וַיָּנָס דֶּרֶךְ בֵּית הַגָּן וַיִּרְדֹּף אַחֲרָיו יֵהוּא וַיֹּאמֶר גַּם אֹתוֹ הַכֻּהוּ אֶל הַמֶּרְכָּבָה בְּמַעֲלֵה גוּר אֲשֶׁר אֶת יִבְלְעָם וַיָּנָס מְגִדּוֹ וַיָּמָת שָׁם :


8) שלושת הערים : ירושלים , תרצה ועין - גנים היו ראויות לשמש כערי ממלכה . ואכן בימי ירושלים של אסא , מלך עמרי בתרצה ותבני בן גינת בעין - גנים . עמרי העדיף לבנות עיר חדשה מאשר לשקם את תרצה השרופה ע " י זמרי , לכן קנה את שומרון , בנה אותה , וכעבור חמש שנים עבר לגור בה כעיר ממשל . שומרון דחתה את תרצה , ניצחה את עין - גנים , והושוותה לירושלים ( אהלה ואהליבה )
