"יְהוָה" מאס בנו...
"יְהוָה" נטש אותנו...
"יְהוָה" לקח מאתנו את כבודנו שהתקיים בזכות "לוחות הברית"...
וכל זאת באשמתם של בני לוי...
הכהנים והלוויים אשר הפרו את בריתם עם "יְהוָה"...ברית "החיים והשלום"...
הכהנים והלוויים אשר הכשילו רבים בתורה...

על-כן..."וַתְּהִי הָאֱמֶת נֶעְדֶּרֶת":

"בָּטוֹחַ עַל-תֹּהוּ וְדַבֶּר-שָׁוְא"
"הָרוֹ עָמָל וְהוֹלֵיד אָוֶן"

"בֵּיצֵי צִפְעוֹנִי בִּקֵּעוּ"
"וְקוּרֵי עַכָּבִישׁ יֶאֱרֹגוּ"
"הָאֹכֵל מִבֵּיצֵיהֶם יָמוּת"
"וְהַזּוּרֶה תִּבָּקַע אֶפְעֶה"
"קוּרֵיהֶם לֹא-יִהְיוּ לְבֶגֶד"
"וְלֹא יִתְכַּסּוּ בְּמַעֲשֵׂיהֶם"

"מַעֲשֵׂיהֶם מַעֲשֵׂי-אָוֶן"
"וּפֹעַל חָמָס בְּכַפֵּיהֶם"

"רַגְלֵיהֶם לָרַע יָרֻצוּ"
"וִימַהֲרוּ לִשְׁפֹּךְ דָּם נָקִי"

"מַחְשְׁבֹתֵיהֶם מַחְשְׁבוֹת אָוֶן"
"שֹׁד וָשֶׁבֶר בִּמְסִלּוֹתָם"

"דֶּרֶךְ שָׁלוֹם לֹא יָדָעוּ"
"וְאֵין מִשְׁפָּט בְּמַעְגְּלוֹתָם"

"נְתִיבוֹתֵיהֶם עִקְּשׁוּ לָהֶם"
"כֹּל דֹּרֵךְ בָּהּ לֹא יָדַע שָׁלוֹם"...
(ישעיה נט')
