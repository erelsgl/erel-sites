מה אשמה סוריה?

אל:

נכתב ב: 03:05:53  16.02.2007, כתוספת/תגובה ל: תגובה על: תגובה על: תגובה על: צדקיהו בבבל

"ירמיהו הנביא נתן את הספר לשריה בן-נריה בן-מחסיה וזאת למען יקרא את כל דברי האלהים שבספר מפי ירמיהו אל יושבי בבל אֶל הַמָּקֹום הַזֶּה לְהַכְרִיתֹו לְבִלְתִּי הֱיֹות בּוֹ יוֹשֵׁב לְמֵאָדָם וְעַד בְּהֵמָה כִּי שִׁמְמֹות עוֹלָם תִּהְיֶה: ועל כן באמת המקום הזה נהפך למדבר [סוריה] שלא ישב בו איש"

סוריה היתה אז בסה"כ שטח שנכבש ע"י הבבלים - לא ממלכת בבל המקורית. אם כך, מה הטעם לקרוא נבואת פורענות, שנכתבה על ממלכת בבל, דווקא בסוריה? ומדוע דווקא סוריה, מכל השטחים שנכבשו ע"י בבל, "זכתה" להפוך למדבר שממה?!

הרבה יותר הגיוני לקרוא את נבואת הפורענות בעיר הבירה, העיר בבל שנמצאת בין הנהרות הגדולים שבעיראק, ואז להשליך את הספר אל הנהר הקרוב לעיר.
