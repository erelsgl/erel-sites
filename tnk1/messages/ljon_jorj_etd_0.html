כתבתי בעבר שיש לדעתי קשר בין עתיד לבין עומד

ומסתבר שיש קשר בין עתיד = עומד = עמוד = עתוד..

השמים / תבל / ארץ מדומים כעומדים על עמודים (עתודים / מוסדות / איתנים / רגליים)
