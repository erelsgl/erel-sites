מורה לתורה משה דין הסופר עבד-יהוה מזרע אהרן הכהן הלוי
ברצוני להזכיר לעדות כי אני עבד-יהוה ככתוב:
בספר ויקרא פרק כה:(מב) כִּי עֲבָדַי הֵם אֲשֶׁר הוֹצֵאתִי אֹתָם מֵאֶרֶץ מִצְרָיִם לֹא יִמָּכְרוּ מִמְכֶּרֶת עָבֶד
וגם בהמשך בפסוק 55 כתוב:(נה) כִּי לִי בְנֵי יִשְׂרָאֵל עֲבָדִים עֲבָדַי הֵם אֲשֶׁר הוֹצֵאתִי אוֹתָם מֵאֶרֶץ מִצְרָיִם אֲנִי יְהֹוָה אֱלֹהֵיכֶם
אך יש עבדי יהוה יראים גם קטנים וגם גדולים כאשר בספר תהילים פרק קטו פסוק 13 כתוב:
(יב) יְהֹוָה זְכָרָנוּ יְבָרֵךְ יְבָרֵךְ אֶת בֵּית יִשְׂרָאֵל יְבָרֵךְ אֶת בֵּית אַהֲרֹן:
(יג) יְבָרֵךְ יִרְאֵי יְהֹוָה הַקְּטַנִּים עִם הַגְּדֹלִים:
ואני עבדו יודע גם כי לא רק אני וכי כל בני ישראל אשר ישמעו בקול יהוה אלהים ככתוב בתורתו תורת משה אשר נתנה לכהנים בני לוי יקרא להם עבדיו כאשר אמר אלהים
