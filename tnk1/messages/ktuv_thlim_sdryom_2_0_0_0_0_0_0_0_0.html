אחרי חודש אתה נזכר להגיב - מה קרה?

וכמובן הס מלהזכיר שאתה כה קבוע בגרסתך שעובדות לא יבלבלו אותך אבל ננסה כמה:

אם הגמרא הייתה באיזור 0 שהיה 70 שנה לפני החורבן זה מותיר לנו כ- 350 שנה לזרא ונחמיה ו- 820 שנה לדוד (הנה מתמטיקה: 350+70+400) אז 1500 השנה שלך פחתו בחצי כמובן שאתה היית שם ואתה יודע טוב מאוד כמה מסורת עברה.

עשה לי טובה אם אין לך משהו חכם להגיד - אל תגיד
"גם אויל מחריש חכן יחשב"
"מי יתן החרש תחרישון ותהי לכם לחכמה".

מכל מקום אם אתה כל כך רוצה לכפור בחז"ל - צר לי עליך אבל אל תנסה לשכנע אותנו עם הסברות הטפשיות שלך.
במיוחד לאור העובדה ששוב ושוב אתה מתגלה בכסלתך ראה מחלוקותיך עם אראל והעקשנות שהייתה לך על גיאוכרונולוגיה.

נ.ב. טרחת לבדוק קצת על זה באנציקלופדיה?
