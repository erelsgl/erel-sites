מאמרים על הקשר בין טעם המצווה לבין הגדרת המצוה

  1. איך לקיים בימינו פרטי מצוות שאיננו יודעים את טעמן? / מחברים שונים -> דרישת האלהים באמצעות חקר המקרא
  2. בניין אב = כללו של דבר = רוח המצוה / אראל ועוד -> עצור כאן חושבים
  3. האם טעם המצוה משפיע על הלכותיה? / מחברים שונים -> עצור כאן חושבים
  4. עיון ("לומדות") בהבנת מהות המצוות; ההבדל בין טעם לבין מהות / יהוסף ועוד -> עצור כאן חושבים
  5. שירה = הרובד הנסתר הקיים בתורה / צחי צויג -> דרישת ה' באמצעות חקר המקרא
